# 🎨 JavaAsciiPatternProject

This project is a **Java-based ASCII Art Generator**.  
It allows you to display characters, words, sentences, and ranges of characters in **ASCII art style** directly in the console.

---

## 📌 Features

- Interactive **menu system** (via `JOptionPane` popups).
- Generate ASCII art for:
  1. 🔤 **Single Character**
  2. ✍️ **Word**
  3. 📝 **Sentence**
  4. 🔢 **Range of Characters**
- Supports:
  - Letters **A–Z**
  - Digits **0–9**
  - Special characters: `@ - _ .`
- Loops back to the menu for multiple runs until the user exits.

---

## 💻 Technologies Used

- **Java (Core)**
- **JOptionPane** for GUI-based user input
- **ASCII Art Matrix Mapping**
